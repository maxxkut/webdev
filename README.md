# webdev

How to run the project:
Download all files 
Install all dependecies in project (npm install)
Start mongoDB server ("C:\..\mongod.exe" -dbpath .. (port 27017))
Start server(npm start)
Start nodemon (nodemon server.js)
go to (http://localhost:3000/)
